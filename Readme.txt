Corresponding peer-reviewed publication

This dataset corresponds to the output files that were produced for the study reported in:

Schwartz et al. Seasonal Bird Migration Explains Regional Synchronicity and Amplification in Human West Nile Virus Case Numbers. TBD.

When utilizing any files from this dataset, kindly cite both the aforementioned article and the dataset itself.
